1.1 Template Filters and How do they work

A template filter is a small, built in function that takes a value, modifies it, and returns a new value for display.
It is applied inside a display bracket using a pipe symbol ( | )
Template Tags control the logic and Template Filters apply the Formatting.

This is a variable value: {{name}}

After applying the upper filter: {{name | upper}}

Applying more than one filter with arguements: {{name | upper | truncatechars:9 }}
