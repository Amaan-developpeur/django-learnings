Django Template Numeric & Date/Time Filters Demo

{% comment %} In your view, make sure you pass: context = { 'num1': 15, 'num2': 4, 'pi_value': 3.141592, 'some_date': datetime(2025, 10, 24, 14, 30), } {% endcomment %}

Numeric Filters

Code: {{ num1|add:10 }}

Output: {{ num1|add:10 }}

Code: {{ num1|divisibleby:5 }}

Output: {{ num1|divisibleby:5 }}

Code: {{ pi_value|floatformat:2 }}

Output: {{ pi_value|floatformat:2 }}

Date & Time Filters

Code: {{ some_date|date:"Y-m-d" }}

Output: {{ some_date|date:"Y-m-d" }}

Code: {{ some_date|time:"H:i:s" }}

Output: {{ some_date|time:"H:i:s" }}

Code: {{ some_date|timesince }}

Output: {{ some_date|timesince }}

Notes:

  • add: adds a number to the variable
  • divisibleby: returns True if variable divisible by given number
  • floatformat: formats float to given decimal places
  • date/time: formats datetime objects
  • timesince: calculates approximate time since given datetime
